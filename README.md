<h1>Pokedex</h1>

<p>Este é um projeto pessoal, ao qual a ideia partiu de um desafio para uma vaga de emprego.</p>

<p>O desafio era produzir uma Pokedex, que nada mais é do que uma espécie semelhante à um Wiki de pokemons, consumindo a API (PokéAPI) e extraindo nome, elemento e tipo, além de adcionar algumas funcionalidades, como cor de fundo correspondente ao elemento do Pokemon, searchbar, barra de navegação para trocar de página ou scroll infinito. Infelizmente o tempo era muito limitado e na época meu conhecimento sobre React também, portanto trouxe a ideia para casa.</p>
